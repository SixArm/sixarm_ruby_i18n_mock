I18nMock is an internationalization localization mock object, good for testing Rails I18n classes.

Example

  require 'rubygems'
  require 'i18n_mock'
  I18n.locale="en-US"
  I18n.locale => "en-US"
  I18n.load_path="lib"
  I18n.load_path => "lib"
